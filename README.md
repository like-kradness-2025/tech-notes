# tech-notes

技術ノートシステム。Discord から疑問を投げると Hermes Agent が調査し、HTML 記事として GitHub Pages に公開します。

## 概要

- ワンショットモード: 「調べてノート: <トピック>」で即時調査→記事化
- 対話モード: 質問→対話で深掘り→「保存して」で会話全体を1つの記事に
- 全記事はダークテーマの静的 HTML、GitHub Pages でホスティング
- 参照URL を各記事に明示

## 運用

運用ルールは `AGENTS.md` を参照。
