まず結論

吸収シグナルは、「売りが強いのに価格があまり下がらない場面」で出やすい。 このタイプは、ちゃんと条件がそろうと 約6割 は勝ちやすい。

ただし、どの場面でも使えるわけではない。 場所・時間・反応の速さ を見て、怪しい場面を捨てるのが大事。

吸収って何？

かんたんに言うと、売りを誰かが受け止めている状態。 みんなが「売りたい」と思っても、価格がスルッと落ちないなら、裏で買っている人がいるかもしれない。

イメージは綱引き。

  • 相手が強く引いている
  • でもロープがあまり動かない
  • → 受け止める側が強い = 吸収の可能性

つまり吸収は、「売りが空振りしている瞬間」 を探す考え方。

6割勝ちやすい場面

勝ちやすいのは、無理のない場所で、自然に支えが入っている場面。 もっと短く言うと、「落ちそうで落ちない」 とき。

1. 価格が“遠すぎない”

平均価格から少しズレているくらいはOK。 でも、遠くまで行きすぎると疲れた相場になりやすい。 そこからの吸収は、無理やり支えているだけになりがち。

2. みんなが見ている場所の近く

VWAPや直近のよく取引された価格の近くは、参加者が多い。 人が多い場所は、買いも売りも入りやすいので、支えが効きやすい。

3. すぐ反応する

本当に強い吸収なら、入ったあとに早めに反応する。 逆に、入ってしばらくしてもダメなら、思ったほど買いが強くない可能性が高い。

4. 変な時間帯ではない

市場は時間帯で性格が変わる。 たとえば東京の昼は、シグナルがだましになりやすかった。 同じ形でも、時間が悪いと勝ちにくい。

まとめると、勝ちやすい吸収は 「場所がよくて、時間もよくて、入ったあとすぐ反応する」 とき。

負けやすい場面

負けるパターンもかなり共通している。 典型的なのは次の3つ。

  • 東京の昼 など、だましが増えやすい時間帯
  • 平均から遠すぎる場所 にいるとき
  • 入ってすぐ悪化する とき

たとえるなら、山の上の細い道で荷物を押している状態。 うまくいくときは少し進むが、失敗するとすぐ転がりやすい。

どう使うか

このシグナルのコツは、当たりやすいものだけ残す こと。 勝率を上げたいなら、無理に回数を増やすより、怪しい場面を捨てるほうが効く。

  1. まず「場所」が変かどうかを見る
  2. 次に「時間帯」が悪くないかを見る
  3. 最後に「入ったあと15分くらいの反応」を見る

もし15分くらいで含み損が続くなら、見切る。 吸収が本物なら、もっと早く答えが出やすい。

実戦チェックリスト

  • 売りが出ているのに、価格があまり下がっていないか
  • VWAPや直近の価格帯から、遠すぎないか
  • 東京の昼みたいな悪い時間帯ではないか
  • 入ったあと、すぐ反応しているか
  • 15分見てダメなら、早めに切るか

要するに、吸収シグナルは「何となく強そう」ではなく、 “売られても落ちない場所” を拾う道具。 そこを外すと負けやすいので、場所・時間・反応 の3点を見るのが大事。
